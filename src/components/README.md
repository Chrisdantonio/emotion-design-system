Design System Components

Folder layout follows the Design System Architect agent rules (BEM, Atomic Design, Semantic HTML).

- atoms/: Basic UI pieces (buttons, inputs, labels).
- molecules/: Composed groups of atoms (search bars, form fields).
- organisms/: Complex sections (headers, footers, layouts).

Before adding components, determine their atomic level and follow BEM naming.
Refer to .github/agents/design-systems-architect-expert-agent.agent.md for conventions.
